📊 Task Manager v1.0 (Python)
Aplikacja desktopowa do zarządzania zadaniami.
✨ Funkcje:

    Interfejs graficzny w CustomTkinter.
    Baza danych oparta na plikach JSON.
    Statystyki postępu obliczane na żywo (pętla for).
    Bezpieczny autozapis przy zamykaniu okna.

🛠️ Technologie:
Python, JSON, Git.
